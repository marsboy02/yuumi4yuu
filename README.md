## venv 관련

```bash
# 가상 환경 생성
$ python3 -m venv venv
```

```bash
# 가상 환경 활성화
$ source venv/bin/activate
```

```bash
# 가상 환경 비활성화
$ deactivate
```

```bash
# requirement.txt를 통한 패키지 설치
$ pip3 install -r requirements.txt
```

```bash
# 패키지 의존성 저장
$ pip3 freeze > requirement.txt
```
